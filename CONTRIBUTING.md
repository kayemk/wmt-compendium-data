# Contributing (Decap Admin only)

Thanks for helping improve the WMT Compendium!

## How to contribute
1. Open the Admin UI:
   https://kayemk.github.io/wmt-compendium-data/admin/

2. Log in with GitHub.

3. Add or edit entries in:
   - **Warbands**
   - **Units**

4. Use the **Source** fields (book name + page) so others can verify the data.

## Rules
- **Do not upload PDFs** or copyrighted material.
- Keep text **short and factual** (no long rulebook excerpts).
- **IDs are stable keys**. Create them once and do not change them later.
- If you are unsure, add a note in the PR description.

## What happens after you save
Your changes will create a Pull Request. A maintainer will review and merge it.

Thank you!
